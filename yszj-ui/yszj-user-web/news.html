科技

新浪科技讯 北京时间6月8日消息，据国外媒体报道，NASA于美东时间6月7日下午两点（北京时间6月8日凌晨2点）举行新闻发布会，宣布了两项关于火星的重要发现：1、好奇号火星漫游车在接近火星地表处一块有30亿年历史的沉积岩中发现了有机分子，说明火星可能曾存在远古生命；2、发现火星大气中的甲烷含量存在季节性波动现象，或与现代火星生命有关。虽然不足以说明火星上一定存在生命，但对未来的火星表面和地下探索任务而言，这些发现无疑是个好兆头。
